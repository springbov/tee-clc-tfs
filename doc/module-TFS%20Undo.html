Module: TFS Undo

Removes pending changes from a workspace.
Required Permissions
To use the undo command, you must be the owner of the specified workspace or have the global Administer workspaces permission set to Allow.
Parameters:
Name Type Description
itemspec Array File(s) and folder(s) to get latest version of. If null/undefined, equals CWD.
options Object Undo command options
viaApi Boolean Is this function called via tfs API ?
